## Notes

To Start Application, Do 2 Things,

1.Start Backend Server, run follwing command in terminal

```bash
yarn server
```

2.Start Frontend Server, run follwing command in terminal

```bash
yarn dev
```

To Build For Deployment

```bash
yarn build
```
